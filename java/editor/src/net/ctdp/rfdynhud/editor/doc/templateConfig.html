This displays the template WidgetsConfiguration ini used for Widget templates.

This file is searched for in the config folder where all the other overlay.ini etc. files are.
It is first searched in the folder of the currently loaded config file and then up to the root config folder. It must be named "templates.ini". If the file is found, it is reloaded any time the editor is opened and when a configuration was loaded.

When a new Widget is added in the editor and a templates.ini is loaded and a Widget of the requested class is found in the among the templates, the Widget is cloned from the template Widget.
